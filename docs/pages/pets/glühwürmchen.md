# Glühwürmchen

<img align="right" width="70" height="75" src="../../../assets/image/pets/Glühwürmchen.png">

Glühwürmchen sind hilfreiche Begleiter für alle, die sich mit dem Gärtnern beschäftigen. Sie bieten exklusive Vorteile bei der Heilkrauternte. Das Glühwürmchen kann ausschließlich durch die [Heilkraut-Kiste](../../pages/lootboxen/heilkrautkiste.md) erworben werden.

## Die verschiedenen Stufen des Glühwürmchens

| Stufe | 1. Fähigkeit | 2. Fähigkeit | 3. Fähigkeit |
|:-:|:-:|:-:|:-:|
| Gewöhnlich | Du hast eine 2 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze |
| Ungewöhnlich | Du hast eine 5 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze | Geht deine Heilkraut-Pflanze kaputt, erhältst du das Heilkraut trotzdem. |
| Selten | Du hast eine 10 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze | Geht deine Heilkraut-Pflanze kaputt, erhältst du das Heilkraut trotzdem. |
| Episch | Du hast eine 15 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze | Geht deine Heilkraut-Pflanze kaputt, erhältst du das Heilkraut trotzdem. |
| Legendär | Du hast eine 20 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze | Geht deine Heilkraut-Pflanze kaputt, erhältst du das Heilkraut trotzdem. | Dein Mohn bleibt dauerhaft in deiner Tasche erhalten. |
| Ultralegendär | Du hast eine 25 % Chance auf einen Lootbox-Drop beim Ernten einer Heilkrautpflanze | Geht deine Heilkraut-Pflanze kaputt, erhältst du das Heilkraut trotzdem. | Dein Mohn bleibt dauerhaft in deiner Tasche erhalten. |
