# Wöchentliche Quests
Jede Woche werden neue Aufgaben erteilt. Diese kann man im Spielermenü (/m oder Offhand-Taste + Shift) einsehen.
## Lootchests
Mit dem erledigen von Aufgaben erhält man Punkte und mit diesen kann man verschiedene Belohnungen abholen. (Zur Zeit gibt es nur eine Belohnung)

<br>

| Belohnung | Punkte |
| :-: | :-: |
| Gewöhnliche Kiste | 50 |
| Heilkraut-Kiste | 70 |

### Gewöhnliche Kiste

| Loot | Seltenheit |
| :-: | :-: |
| 64x Brot | Gewöhnlich |
| 64x Wasser | Gewöhnlich | 
| 300 EXP | Ungewöhnlich |
| 250€ Bargeld | Ungewöhnlich |
| 500 EXP | Ungewöhnlich |
| Brechstange | Ungewöhnlich |
| 750€ Bargeld | Selten |
| Dietrich | Episch |
| Simon's Schlüssel | Legendär |

!!! info Simon's Schlüssel
    • Erhältlich durch das Abschließen von Jobrotations oder Lootchests.
    •  Ermöglicht es, jederzeit eine Schwarzmarkauktion zu betreten.
    • Verschwindet nach einmaliger Nutzung.
