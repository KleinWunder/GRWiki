# Winzer (Nebenjob)

Der Winzer ist ein Nebenjob, welcher sich am Weinberg befindet. 
(**/navi Winzer**)


| Bushaltestelle | Farm |
| :-: | :-: |
| Navi | /navi Weinberg |
| Dauer | ca. 2 Minuten|
| Cooldown | 5 Minuten |
| Gewinn | Geld, XP, [Farming-XP](../../pages/skills/farming.md) |


## Aufgabe
Ernte Weintrauben von den Weinreben
